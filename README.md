- 具体流程图
![image](github-pull-request.png)
### 具体步骤
1. 首先，在Github页面上，点击“Fork”，将iCode的Git库拷贝到自己账户
2. 然后，对拷贝的Git库进行修改，将内容变更提交到自己的账户
3. 最后，在Github页面上，点击"New pull request"，向iCode官方发起代码审阅
4. 希望和大家共建和谐美好开源环境！
